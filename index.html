Qué hace esta skill

🔍

Rutas inteligentes

Busca por capas: directo desde BIO, vía Madrid o Barcelona, y vía hubs europeos (París, Ámsterdam, Londres, Frankfurt, Lisboa).

🌍

Destinos lejanos incluidos

Para América, Asia, África u Oceanía, encuentra la combinación más barata de vuelo corto + vuelo de largo radio.

📊

Gráfico de evolución

Cada consulta guarda el precio. Con el tiempo genera un gráfico para ver si el precio está por encima o por debajo de lo habitual.

🟢

Veredicto claro

Verde si está barato, amarillo si está en rango normal, rojo si está caro. Sin ambigüedades.

⚠️

Contexto ciudad pequeña

Compara siempre el precio desde Bilbao con lo que costaría desde Madrid, para saber si vale la pena ir a un aeropuerto más grande.

🔥

Chollos de la semana

Sin destino fijo, muestra los vuelos más baratos disponibles ahora mismo desde Bilbao.

Cómo instalarlo

1

Instala Claude Code

Abre VS Code, ve a Extensions y busca Claude Code. Necesitas una cuenta de Anthropic.

2

Descarga este kit

Haz clic en "Descargar kit" arriba, descomprime la carpeta y ábrela en VS Code con Archivo → Abrir carpeta.

3

Escribe tu destino

Abre el panel de Claude Code y escribe algo como ¿vuelos baratos a Bangkok en agosto? La skill se activa automáticamente.

4

Consulta varias veces

Cada consulta queda guardada en tu historial local. Cuanto más la uses, más preciso se vuelve el análisis de si el precio está barato o caro.
