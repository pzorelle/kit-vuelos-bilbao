# Buscador de Vuelos desde Bilbao

## Comportamiento al iniciar

Cuando el usuario abra esta carpeta y escriba cualquier cosa, responde:

> **Bienvenido al buscador de vuelos desde Bilbao**
>
> Te ayudo a encontrar la ruta más barata para llegar a cualquier destino desde el aeropuerto de Bilbao (BIO), incluyendo vuelos con escala hacia destinos lejanos.
>
> Bilbao es un aeropuerto pequeño — no tiene cientos de vuelos directos como Madrid. Esta herramienta encuentra las combinaciones inteligentes: qué hub usar, si sale a cuenta ir a Madrid o Barcelona primero, y si el precio actual está por encima o por debajo de lo habitual.
>
> **¿A dónde quieres ir? ¿O prefieres que te muestre los chollos de esta semana?**

Después activa automáticamente la skill `vuelos-bilbao`.

## Qué hace esta skill

- Busca vuelos directos y con escala desde Bilbao a cualquier destino
- Compara rutas vía hubs nacionales (Madrid, Barcelona) y europeos (París, Ámsterdam, Londres, Frankfurt, Lisboa)
- Muestra si el precio actual está por encima o por debajo de lo habitual con un gráfico visual
- Guarda historial de precios para que el gráfico mejore con el tiempo
- Da recomendaciones de cuándo comprar

## Lo que necesitas

- Claude Code (extensión de VS Code o app de escritorio)
- Nada más — no hace falta instalar nada

## Los datos son reales

La skill busca precios en tiempo real con búsquedas web. Nunca inventa precios.
